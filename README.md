# Digital Brutalist Portfolio

A unique, experimental personal portfolio with an anti-design aesthetic.

## Features

- High-contrast neon color scheme (green on purple)
- Glitchy, non-standard layout with rotated and skewed elements
- Custom cursor effect
- Monospaced typography
- Responsive design

## Files

- `index.html` - Main HTML file
- `style.css` - Styling

## Usage

Open `index.html` in your web browser to view the portfolio.
